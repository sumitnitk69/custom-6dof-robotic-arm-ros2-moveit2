# Custom 6-DOF Robotic Arm

Open-source custom robotic arm designed in SolidWorks for future ROS2, Gazebo, RViz, MoveIt2, and real hardware integration.

## Features
- 6-DOF architecture
- MG995 + MG90S servo design
- Lightweight wrist mechanism
- 3D printable structure
- Future ROS2 support
- Open-source STL files

## Current Progress
- Base completed
- Shoulder and elbow completed
- Wrist pitch and wrist roll completed
- Gripper integrated

## Future Goals
- URDF/Xacro generation
- Gazebo simulation
- MoveIt2 integration
- Physical prototype
- Inverse kinematics
- Camera/OpenCV integration
